HTML

  • The head element contains information about the webpage.
  • The body element represents the visiable content shown to the user.
  • The base element specifies the base URL to use for all relative URLs in a document.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout -b feature/starter-code: Creates a new branch to work on the project without affecting main. -b command lets you name your personal branch
  • git add .: add your changes to branch
  • git add-a: will save the changes we made after doing File>Save on vs code. *Remember to save code or have auto save on*
  • git commit: in git, a commit is a snapshot of your repo at a specific point in time
  • git commit -m:
  • git checkout main: switches to main branch
  • git pull origin main: pulls all new changes from repo to personal main branch
  • git checkout branchname: enters the branch to navigate to
  • git branch --delete branchname
